# Accessible Form UI

This project is a solution for the [Accessible Form UI](https://roadmap.sh/projects/accessible-form-ui) challenge from [roadmap.sh](https://roadmap.sh). The goal was to build an accessible, responsive form UI using semantic HTML and CSS.

- **Semantic HTML**: Uses proper HTML5 elements (`<main>`, `<section>`, `<form>`, `<label>`, `<fieldset>`) for structure and meaning.
- **Accessibility (a11y)**:
  - ARIA attributes (`aria-required`, `aria-invalid`, `aria-describedby`) for screen readers.
  - Clear focus states for keyboard navigation.
  - Sufficient color contrast (WCAG AA compliant).
- **Responsive Design**: Adapts layout for different screen sizes (Flexbox).
- **Theme**: Modern Dark Blue theme with the 'Inter' typeface.
- **Static UI**: Pure HTML/CSS implementation focusing on structure and styling.

# Tabs Component

A simple, reusable tabs component built with HTML, CSS, and JavaScript. It features a modern dark blue theme and smooth content transitions.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Animations)
- Vanilla JavaScript (DOM Manipulation)

## Usage

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  Click on the tabs to navigate between content sections.

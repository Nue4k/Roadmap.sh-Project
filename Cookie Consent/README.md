# Cookie Consent

A simple, customizable cookie consent banner component. It appears at the bottom-right of the screen and allows users to accept cookies.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Fixed Positioning)
- Vanilla JavaScript (Event Listeners)

## Usage

1.  Clone the repository.
2.  Open `index.html` in your browser.
3.  Click "Accept" to dismiss the banner.
4.  Refresh the page to see it again (persistence is disabled for demonstration).

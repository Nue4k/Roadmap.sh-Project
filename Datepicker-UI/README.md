# Datepicker UI

![Datepicker UI Project](https://roadmap.sh/assets/images/projects/datepicker-ui.png)

## Description

This project is a solution for the [Datepicker UI](https://roadmap.sh/projects/datepicker-ui) challenge on Roadmap.sh. The goal is to create a static datepicker component using only HTML and CSS, focusing on layout, positioning, and styling.

## Features

- **Static UI**: A non-functional, visual representation of a datepicker.
- **Custom Styling**: Clean and modern design using CSS variables.
- **Responsive Layout**: Centered layout with a responsive container.
- **Icons**: Integration of Phosphor Icons for a polished look.
- **No JavaScript**: Pure HTML and CSS implementation.

## Tech Stack

- HTML5
- CSS3 (Flexbox, CSS Grid, Variables)
- Phosphor Icons
- Google Fonts (Inter)

## How to View

1. Clone the repository.
2. Navigate to the `Datepicker-UI` folder.
3. Open `index.html` in your browser.

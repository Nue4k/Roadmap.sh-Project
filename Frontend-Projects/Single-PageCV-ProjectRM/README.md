# Single-Page CV
https://roadmap.sh/projects/single-page-cv

A simple, accessible single-page curriculum vitae (CV) template using HTML and CSS.

**Project Page URL:** https://github.com/Nue4k/Single-Page-CV

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/Nuue4k/Single-Page-CV.git
    ```
2.  Navigate to the project directory.
3.  Open `Single-PageCV.html` in your preferred web browser.

# Basic HTML Website
https://roadmap.sh/projects/basic-html-website

A multi-page personal portfolio website built with semantic HTML and CSS. Features a home page with project highlights, separate project listings, articles, and a contact form.

**Project Page URL:** https://github.com/Nue4k/Basic-HTML-Website

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/Nuue4k/Basic-HTML-Website.git
    ```
2.  Navigate to the project directory.
3.  Open `index.html` in your preferred web browser.
